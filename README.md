 🎖️ Sistema de Permuta Militar

Sistema digital para automação de solicitações de permuta de serviço, focado em agilidade, transparência e controle do Comando.

## 🚀 Funcionalidades
- **Solicitação Digital:** Formulário rápido para solicitante e substituto.
- **Painel de Controle (Gabinete):** Tela exclusiva para o Comandante homologar ou negar pedidos.
- **Acompanhamento em Tempo Real:** Status com carimbos visuais (✅ ACEITO / ❌ NEGADO) para os militares.
- **Notificação:** Sistema de monitoramento automático de despacho.

## 🛠️ Tecnologias Utilizadas
- **Backend:** Node.js com Express
- **Banco de Dados:** PostgreSQL (Supabase) / SQLite (Local)
- **Frontend:** HTML5, CSS3 e JavaScript (Vanilla)
- **ORM:** Sequelize
